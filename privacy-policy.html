개인정보처리방침

GEO Analyzer Chrome Extension · 시행일: 2026-03-17

1. 수집하는 정보

GEO Analyzer는 개인정보를 수집하지 않습니다.

  • 사용자 계정, 이메일, 이름 등 개인 식별 정보를 수집하지 않습니다.
  • 브라우징 기록, 방문 URL, 쿠키 등을 수집하거나 외부로 전송하지 않습니다.
  • 분석 데이터는 사용자의 브라우저 내에서만 처리되며 어떤 서버로도 전송되지 않습니다.

2. 데이터 처리 방식

  • 페이지 분석: 현재 탭의 JSON-LD, 메타태그, 헤딩 구조를 읽어 분석합니다. 이 데이터는 팝업이 열려 있는 동안에만 메모리에 존재하며 저장되지 않습니다.
  • robots.txt 확인: 분석 대상 사이트의 robots.txt를 읽어 AI 크롤러 접근 허용 여부를 확인합니다. 이 데이터는 외부로 전송되지 않습니다.
  • 규칙 갱신: GitHub Pages에서 분석 규칙 파일(rules.json)을 주기적으로 다운로드합니다. 이 요청에 사용자 식별 정보는 포함되지 않습니다.
  • 로컬 캐시: 다운로드된 규칙 파일은 chrome.storage.local에 24시간 캐시됩니다. 개인정보는 포함되지 않습니다.

3. 제3자 제공

GEO Analyzer는 어떠한 데이터도 제3자에게 판매, 전송, 공유하지 않습니다.

4. 권한 사용 목적

  • activeTab: 현재 탭의 페이지 내용을 분석하기 위해 사용합니다.
  • scripting: 페이지에서 JSON-LD 데이터를 추출하는 content script를 실행하기 위해 사용합니다.
  • storage: 분석 규칙 캐시를 저장하기 위해 사용합니다.

5. 변경 사항

본 개인정보처리방침이 변경되는 경우 이 페이지를 통해 공지합니다.

Privacy Policy

GEO Analyzer Chrome Extension · Effective: 2026-03-17

1. Information We Collect

GEO Analyzer does not collect any personal information.

  • We do not collect user accounts, emails, names, or any personally identifiable information.
  • We do not collect or transmit browsing history, visited URLs, or cookies.
  • All analysis data is processed locally within your browser and is never sent to any server.

2. How Data Is Processed

  • Page Analysis: Reads JSON-LD, meta tags, and heading structure from the current tab. This data exists only in memory while the popup is open and is not stored.
  • robots.txt Check: Reads the analyzed site's robots.txt to check AI crawler access permissions. This data is not transmitted externally.
  • Rules Update: Periodically downloads analysis rules (rules.json) from GitHub Pages. No user-identifying information is included in these requests.
  • Local Cache: Downloaded rules are cached in chrome.storage.local for 24 hours. No personal information is included.

3. Third-Party Sharing

GEO Analyzer does not sell, transfer, or share any data with third parties.

4. Permission Usage

  • activeTab: Used to analyze the content of the current tab.
  • scripting: Used to execute content scripts that extract JSON-LD data from pages.
  • storage: Used to cache analysis rules locally.

5. Changes

Any changes to this privacy policy will be posted on this page.
